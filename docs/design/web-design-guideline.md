# Web 設計規範（Web Design Guideline）

> 由 **ui-designer** 維護。所有介面實作（frontend）必須遵守本規範；規範不足時先擴充規範再實作，不做一次性特例。
> clone 模板後由 ui-designer 在第一個介面任務時填入具體數值。

## 1. 設計原則

1. **三態齊備**：每個互動都定義正常、載入/等待、錯誤三種狀態。
2. **回饋即時**：任何操作 100ms 內要有視覺回饋，即使結果還沒回來。
3. **一致優先**：同一種操作在所有頁面長得一樣；新頁面先套既有元件，不夠再擴充元件庫。
4. **可及性**：文字對比度至少 4.5:1；鍵盤可操作；不用顏色作為唯一的資訊管道。
